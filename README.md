# Art Institute of Chicago - Artworks Table

A React + PrimeReact app that displays artworks from the Art Institute of Chicago API.

### Features
- Server-side pagination
- Multi-row selection
- Custom row selection using OverlayPanel
- Selection persists across pages

### Tech Stack
- React
- TypeScript
- PrimeReact
- Vite
